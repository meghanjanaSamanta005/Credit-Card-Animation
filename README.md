## ‚ú® Features

* **Real-time Updates:** Card details (Number, Name, Expiry) appear on the graphic instantly.
* **Auto-Formatting:** Automatically adds spaces to the card number (e.g., `4242 4242...`).
* **Interactive UI:** Focus animations and gradient backgrounds.
* **Responsive:** Works on desktop and mobile screens.
